# Future Roadmap

P0 검증 이후에만 고려할 기능입니다.

## P1

- 주간 화장대 리포트: 로컬 저장 제품 변화와 사용 완료 상태를 요약합니다.
- 제품 사용기한 메모: 알림 없이 로컬 메모와 정렬만 제공합니다.
- 더 넓은 성분 지식베이스: 공개적으로 재사용 가능한 데이터만 수동 검토 후 추가합니다.

## P2

- 실제 계정 시스템: 서버 인증, 개인정보 처리, 신고/차단 정책이 준비된 뒤 구현합니다.
- 실제 결제: 결제 SDK, 환불 정책, 구독 상태 동기화가 준비된 뒤 구현합니다.
- 실제 제휴 링크: 외부 링크 고지, 추적 동의, 정산 정책이 준비된 뒤 구현합니다.

## 구현하지 않을 항목

- 피부 사진 분석
- 피부 질환 진단
- 트러블 확률 예측
- EWG 또는 외부 사이트 스크래핑
- 실제 제품 인터넷 검색 자동화
- 푸시 알림
- 날씨·자외선 연동
- 관리자 페이지
